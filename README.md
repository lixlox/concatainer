# concatainer

### Install rust if you don't have it (Windows with WSL / linux)
```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

## Installation 

In a terminal : 
```bash
git clone https://github.com/lixlox/concatainer.git
```

```bash
cd concatainer
```

```bash
cargo run "test_case.json"
```
